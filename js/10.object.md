## Задачи по теме 10. Object ##

Дан объект user, выполнить задачи 1-6:

```
const user = {
    name: "Peter",
    age: 33,
    job: "Developer",

    "my address": {
        city: "Moscow",
        country: "Russia",
        abroad: ["Vienna", "Paris", "London", "Miami"]

    item: ["bag", "book", "pencil"]
};
```

### 2. Задача

### 3. Задача

### 4. Задача
